# XXChaCha20

ChaCha20 with a 224-bit nonce. It works the same way as [XChaCha20](https://datatracker.ietf.org/doc/html/draft-irtf-cfrg-xchacha#section-2.3), except there's no `\x00\x00\x00\x00` nonce prefix.
